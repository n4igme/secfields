### 11. Endpoint & Device Security

Covers staff, branch networks, ATMs, etc.

- Endpoint protection (EDR/EPP): Detection and prevention on all devices
- Hardening profiles: Security configurations for different device types
- Mobile device management (MDM): Control and monitoring of mobile devices
- ATM and POS security considerations: Specialized security for financial terminals
- BYOD policy: Bring Your Own Device security requirements
- Patch management: System update and vulnerability remediation
- USB and removable media restrictions: Control of data transfer devices
