### 5. Security Engineering

Building security into systems and infrastructure.

- Secure software development (SSDLC): Security integration throughout development lifecycle
- Secure coding guidelines: Standards and best practices for developers
- SAST/DAST/IAST: Static, dynamic, and interactive application security testing
- Dependency and supply chain scanning: Third-party component security validation
- Infrastructure hardening: Configuration standards for systems and services
- Container and Kubernetes security: Orchestration platform protection
- Secrets management: Secure handling of credentials and sensitive data
- Key management & HSM: Hardware security module integration and operations
- API security: Authentication, authorization, and protection measures
- Configuration baselines: Standard security configurations for systems
